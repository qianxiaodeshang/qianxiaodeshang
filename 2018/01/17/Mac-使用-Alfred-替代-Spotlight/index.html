Mac 使用 Alfred 替代 Spotlight

发表于 2018-01-17 | 更新于 2018-01-18 | 分类于 mac
安装Alfred

先试试免费版本，Alfred官网

关闭spotlight
  • 禁用spotlight服务
    sudo mdutil -a -i off
  • 删除菜单栏spotlight图标
Spotlight恢复方法
  • 启用spotlight服务，
    sudo mdutil -a -i on
  • 恢复菜单栏图标

然后在活动监视器退出 SystemUIServer 进程，或者注销再登陆。

配置Alfred

  • Webserch搜索设置

    • 添加百度搜索：
      链接：https://www.baidu.com/s?ie=utf-8&f=8&wd={query}
      title：百度
      keyword：bd # 自行设置调用关键词。输入bd XXX 即可用百度搜索
      设置图标：复制百度icon图片到右边方框内

    • 淘宝搜索
      Url：https://s.taobao.com/search?oe=utf-8&f=8&q={query}
      title：淘宝
      keyword：tb
      设置图标：复制淘宝icon图片到右边方框内

    • github
      url：https://github.com/search?utf8=%E2%9C%93&q={query}
      title：github
      keyword：github
      设置图标：复制github的icon图片到右边方框内

  • 设置默认搜索引擎为百度
    Features > Default Results > Fallbacks >Setup fallbacks results
    点击右下角 + > Custom Search > 选择刚添加的百度 > 保存完成！
